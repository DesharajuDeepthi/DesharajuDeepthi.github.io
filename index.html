CallLens
Identity-aware call analytics platform
Problem B2B call insights were locked behind SQL — executives and CSMs couldn't self-serve role-appropriate answers from 100 transcripts.
Built 6-node LangGraph pipeline with human-in-the-loop gates → Postgres with Row-Level Security → MCP server serving role-scoped answers to any LLM client (Claude Desktop, Codex). JWT claims thread through every tool call.
Result ~91% of queries handled by rule-based logic; LLM only for ambiguous cases. Total LLM spend under $1 across the corpus.

Research Copilot
Self-correcting retrieval agent over academic papers
Problem Standard RAG confidently hallucinates citations — researchers can't trust answers not grounded in real retrieved papers.
Built Self-correcting loop: grade → rewrite → live-API fallback → synthesize → verify. Hybrid BM25 + vector search with Reciprocal Rank Fusion on Qdrant.
Result Zero hallucinated citations by construction — deterministic citation verification as a hard gate, not a prompt instruction.
